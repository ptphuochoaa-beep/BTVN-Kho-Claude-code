Tải file & phân tích

Hỗ trợ file Excel (.xlsx) và CSV xuất trực tiếp từ Google Search Console

💡
Khuyên dùng: xuất Excel (.xlsx) — GSC gộp toàn bộ báo cáo (Từ khoá, Trang, Quốc gia, Thiết bị...) vào 1 file duy nhất, và nếu bạn bật So sánh 2 khoảng thời gian trước khi xuất, file Excel đó tự có sẵn cả 2 kỳ để công cụ so sánh ngay — không cần upload 2 lần.
Nếu xuất CSV, GSC sẽ tách thành nhiều file rời (Queries.csv, Pages.csv, Devices.csv…), mỗi file chỉ chứa 1 kỳ và không tự so sánh được — vẫn dùng được, chỉ cần chọn tất cả các file CSV đó cùng lúc ở ô bên dưới.

Kéo thả file vào đây

📅 Dữ liệu kỳ để so sánh (kỳ trước đó)

Kỳ trước đó (tuỳ chọn)

1 file Excel hoặc nhiều file CSV của kỳ cần so sánh

Chỉ áp dụng cho những loại báo cáo (Từ khoá/Trang/...) mà file chính chưa có sẵn 2 kỳ.

Mỗi dòng 1 quy tắc: pattern=Tên nhóm. URL nào khớp pattern sẽ được gộp vào nhóm đó; URL không khớp sẽ tự gộp theo thư mục đầu tiên trong đường dẫn.
